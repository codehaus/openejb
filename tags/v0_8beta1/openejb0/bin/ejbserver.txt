Usage: openejb start ejbserver [options]

OPTIONS:

  -p <port> Starts the OpenEJB EJB Daemon on the port specified. 
            Default port is 4201.
  
  -h <host> Binds the OpenEJB EJB Daemon to the specified host address.     
            Default is 127.0.0.1.
  

The OpenEJB EJB Container System, a local EJB API and Server.
For OpenEJB updates and additional information, visit
http://openejb.sf.net
