The classes and interfaces directly under this package define the HTTP Web Administration Console.

Package Specification

The Web Administration Console is an HTTP console to perform various administrative tasks including:

  • EJB Deployment
  • EJB List
  • List of Log files
  • Graphical Configuration


...and more to come! This Administration console will premere in the 1.0 release of OpenEJB.
