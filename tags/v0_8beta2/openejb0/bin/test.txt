Usage: openejb test [options]

OPTIONS:

  intra-vm  Default. Runs the test suite against OpenEJB in 
            embedded library mode using the IntraVm Server
  
  ejbserver Runs the test suite against OpenEJB in client/server
            mode using the OpenEJB Server.                   
            
  corba     Runs the test suite against OpenEJB in client/server
            mode using the OpenEJB CORBA Server with OpenORB

  help      Display this message.
