Usage: openejb start [options]

OPTIONS:

  intra-vm  Displays information on the IntraVM Server
  
  ejbserver Default. Starts OpenEJB in client/server
            mode using the OpenEJB Server and OpenEJB's
            own protocol.
  
  corba     Starts OpenEJB in client/server mode using 
            the OpenEJB CORBA Server with OpenORB 
            and the RMI-IIOP protocol.


The OpenEJB EJB Container System, a local EJB API and Server.
For OpenEJB updates and additional information, visit
http://openejb.sf.net
